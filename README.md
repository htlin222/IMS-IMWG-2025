# IMS-IMWG 2025 Consensus Genomic Staging in NDMM

A [Slidev](https://sli.dev/) presentation covering **Maclachlan et al., Blood Advances 2025** — the IMS-IMWG Consensus Genomic Staging system predicting outcomes with daratumumab-based quadruplet regimens for newly diagnosed multiple myeloma (NDMM).

## Tech Stack

- [Slidev](https://sli.dev/) — Presentation framework for developers
- [Vue 3](https://vuejs.org/) — UI components
- [Chart.js](https://www.chartjs.org/) / [vue-chartjs](https://vue-chartjs.org/) — Data visualization
- [Lucide Icons](https://lucide.dev/) — Icon set

## Getting Started

```bash
pnpm install
pnpm dev
```

## Available Scripts

| Script        | Description                          |
| ------------- | ------------------------------------ |
| `pnpm dev`    | Start the Slidev dev server          |
| `pnpm build`  | Build the presentation for production |
| `pnpm export` | Export slides to PDF                 |
